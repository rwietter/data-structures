## 🚀 Quick start

1.  **Start developing.**

    Clone the project with git clone and

    Navigate into your new site’s directory and start it up.

    ```shell
    cd css-drawing-starship/
    code .
    ```

    Open with live server...

    Your site is now running at `http://localhost:8000`!
